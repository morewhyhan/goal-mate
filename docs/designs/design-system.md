# Design System

Goal Mate 的设计理念是低熵极简主义：只保留能降低用户决策负担的信息和操作。

## Core Principles

- AI 处理复杂框架，用户只看下一步。
- 用成熟管理方法组织后台结构：OKR、WBS、Gantt、Milestone、Status、Review。
- 页面用强隐喻解释功能，而不是靠长文说明。
- Goals 只读；Logs 承接系统产出的可编辑 Markdown 文件；Agent 是对话入口；Settings 配置长期权限规则。

## Page Metaphors

- Today = 四象限 + 下一步卡片。
- Goals = OKR 摘要 + WBS 甘特图。
- Logs = nested file tree + Markdown editor.
- Agent = conversation history + Codex-style chat.
- Settings = category sidebar + focused setting detail.

## Visual Rules

- 少文字，强结构。
- 状态和颜色必须服务判断。
- 按钮只放在可操作页面：Today、Agent、Settings。Goals 不放按钮。
- 甘特图表达执行计划，不直接承载所有 OKR 解释。
