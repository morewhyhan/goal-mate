## 4. 风险清单

| 风险 | 说明 | 处理方式 |
| --- | --- | --- |
| 页面视觉仍可能不符合直觉 | HTTP 200 和 build 不能证明视觉体验 | 做浏览器截图/人工验收 |
| 服务器长期运行未证明 | 本地运行通过不等于电脑关闭后也可用 | 执行 self-hosted runtime verification |
| QQ 平台限制可能影响主动 C2C 消息 | 本地 Scheduler/QQ 代码存在不等于平台允许全部发送 | 服务器上记录 SchedulerEvent 成功/失败原因 |
| reset 未验证 | 只验证了 migrate deploy，不代表 reset 能从零初始化 | 单独执行 reset 验收，需用户确认可清库 |
