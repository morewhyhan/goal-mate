## 6. 注意事项

- `pnpm build` 不应在 `pnpm dev` 运行时并行执行。Next 的 `.next` 产物可能被生产构建覆盖，导致运行中的 dev server 出现 chunk 缺失。正确顺序是：先 build，之后重新启动 dev server，再执行运行时验收。
- 当前模型连接测试是本地占位接口，验证配置存在、接口可调用、密钥脱敏；没有真实调用 DeepSeek 外部 API。
