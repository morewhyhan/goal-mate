| Issue | Fix |
| --- | --- |
| Logs API typecheck failed because `frontmatter` could be `null` | Normalized `frontmatter` to `Record<string, unknown> | undefined` before calling Markdown store |
| `/api/settings/export` returned 500 on DBs missing new runtime tables | Applied existing migration `20260702000500_agent_tools_scheduler` |
| Agent Loop verifier could not read source files from `src/package.json` working directory | Resolved verifier file reads relative to the script app root |

| Scheduler required waiting for a real clock minute to verify reminders | Added `--once --force-reminder=morning_planning` and `pnpm worker:scheduler:once` |
